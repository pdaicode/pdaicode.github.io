Deep Dream - Inceptionism (Ubuntu + mutiple version Anaconda Python 2.7/3.4)

Tweet

Also available at https://sites.google.com/site/declanide/project-blog/deepdream-inceptionism

System: Ubuntu (dual boot with Windows 7), Python (multiple version with Anaconda), sample code from Google at https://github.com/google/deepdream

To compile the Python and MATLAB wrappers do make pycaffe and make matcaffe respectively. Be sure to set your MATLAB and Python paths in Makefile.config first!

Distribution: run make distribute to create a distribute directory with all the Caffe headers, compiled libraries, binaries, etc. needed for distribution to other machines.

make pycaffe

make distribute

General instruction for Python:

The main requirements are numpy and boost.python (provided by boost). pandas is useful too and needed for some examples. You can install the dependencies with

for req in $(cat requirements.txt); do pip install $req; done

but we suggest first installing the Anaconda Python distribution, which provides most of the necessary packages, as well as the hdf5 library dependency. To import the caffe Python module after completing the installation, add the module directory to your $PYTHONPATH by export PYTHONPATH=/path/to/caffe/python:$PYTHONPATH or the like. You should not import the module in the caffe/python/caffe directory! Caffe's Python interface works with Python 2.7. Python 3.3+ should work out of the box without protobuf support. For protobuf support please install protobuf 3.0 alpha (https://developers.google.com/protocol-buffers/). Earlier Pythons are your own adventure.

Because Anaconda does NOT work with $PYTHONPATH, we have to manually add caffe to the directories (you can also change the way Anaconda link different packages. I don't know how. If you succeed in changing the link, please send me an email at peng.dai.ca@ieee.org).

My default Anaconda installation is Python3.4. Create a 2.7 environment by

conda create -n py27 python=2.7 anaconda

Then activate the py27 environment by

source activate py27

Please note: caffe is compatible with Python 3.4. However, the sample code for Deep Dream from Google used some of the 2.7 only package. Therefore, for simplicity we go for 2.7. After make pycaffe & make distribute, the caffe-master (follow git) directory should contain a folder called /distribute. It contains 4 subfolders /bin /include /lib /python

change the /python folder to /caffe (change name). This is our python version of caffe.

Copy the /caffe folder to /home/anaconda3/envs/py27/lib/python2.7/site-packages/

Copy the files in /distribute/lib into /home/anaconda3/envs/py27/lib/

For Python 3.4, just change the /distribute to /caffe. Then copy the new /caffe folder to home/anaconda3/lib/python3.4/site-packages

Now everything is ready for the sample code at https://github.com/google/deepdream

Note: protobuf can be installed by conda install protobuf
